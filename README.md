# Sales Prediction using Python

## Project Overview

This project predicts product sales based on advertising expenditures across different media platforms such as TV, Radio, and Newspaper.

## Objective

To analyze the relationship between advertising spending and sales performance and build a predictive model for future sales forecasting.

## Dataset

The dataset includes:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Machine Learning Model

Linear Regression

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Train-Test Split
6. Model Training
7. Sales Prediction
8. Performance Evaluation

## Results

The model successfully predicts sales using advertising expenditure data and identifies the impact of different advertising channels on sales outcomes.

## Conclusion

This project highlights the importance of data-driven marketing decisions and demonstrates how machine learning can assist businesses in forecasting sales performance.
